# SistDistT1

## Ambiente Virtual
Criando ambiente virtual.
```
python -m venv venv
```

Ativar ambiente.
Linux:
```
source venv/bin/activate
```

Windows:
```
.\venv\Scripts\activate
```

Sair do ambiente:
```
deactivate
```

## Instalação do RPyC:
```
pip install rpyc
```

## Execução do servidor:
```
pyhton3 server.py
```

## Execução do cliente:
```
python3 cliente.py 0.0.0.0
```

ou

```
python3 cliente.py localhost
```
